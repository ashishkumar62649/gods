## Rebuilt Instead Of Blindly Copied

- Live aircraft rendering now uses a v2 `LiveFlightRenderer` with fixed-record binary transport and JSON fallback instead of the v1 monolithic telemetry renderer.
- General intelligence points are loaded through `LiveIntelEntityRenderer` and `layerStore` rather than app-mode mock entities.
- Domain tables were added in `0007_v2_operational_domain_tables.sql` instead of forcing all future data into the older public-schema MVP tables.
- Python hazard pipeline now uses USGS Earthquake no-auth fetch, normalize, validate, and optional DB load instead of the old JS-only pipeline path.

## External Blockers

- Space-Track: satellite TLE refresh requires credentials.
- AISStream and Global Fishing Watch: maritime live data requires API keys/tokens.
- NASA FIRMS production fire feeds: MAP_KEY required for many endpoints.
- OpenAQ: public-apis catalog currently lists API key requirement.
- OpenWeatherMap/weather raster tiles: API key required; RainViewer remains no-auth fallback candidate.
