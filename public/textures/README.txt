Drop an equirectangular Earth texture here as `earth_day.jpg`.

Recommended:
  - 2048 x 1024 (or 2048 x 1024 power-of-two)
  - JPEG, ~300-500 KB
  - Public-domain source: NASA Blue Marble
    https://visibleearth.nasa.gov/images/57735/the-blue-marble-land-surface-ocean-color-and-sea-ice
    (pick the 2K version; rename to earth_day.jpg)

The app expects the file at:
  public/textures/earth_day.jpg
which is served at:
  /textures/earth_day.jpg
