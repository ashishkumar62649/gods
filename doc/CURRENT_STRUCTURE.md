# God Eyes Project Skeleton Structure

This document provides a comprehensive overview of the `god eyes` project structure, detailing every major file and folder.

## Root Directory
The root directory contains high-level configuration, orchestration scripts, and documentation.

- `god eyes/`
    - `.agent/` - Agent configuration and metadata.
    - `.cdsapirc` - API configuration for Copernicus Data Space.
    - `.codex/` - Codex-specific data and logs.
    - `.git/` - Git version control history.
    - `.gitattributes`
    - `.gitignore`
    - `adsblol-home.html`
    - `copernicus_land_clms.json`
    - `doc/` - Project documentation and master files.
    - `docker-compose.yml` - Container orchestration.
    - `explorer/` - The primary application (Frontend + Backend).
    - `graphify-out/` - Output directory for data visualization graphs.
    - `node_modules/` - Root level npm dependencies.
    - `package-lock.json`
    - `plugins/` - Custom extensions.
        - `caveman/`
    - `proxy-8788.out.log`
    - `proxy-8790.out.log`
    - `tmp-adsblol-home.html`
    - `tmp-adsblol-script.js`
    - `venv/` - Python virtual environment for backend tasks.
